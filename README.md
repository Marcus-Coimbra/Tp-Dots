# MRT-Dots in C++/SFML

A recriação do jogo Dots2 feito em c++ com o uso da biblioteca SFML.
Este foi um jogo feito como atividade no CEFET-MG/Contagem.

## Instalação

Para compilar, foi utilizado SFML2.1, que foi instalado manualmente no projeto.
